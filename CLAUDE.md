# CLAUDE.md — 岡潔の世界

## このAppが何者か

岡潔の言葉を、ただそこに置く。
装飾のためにデザインするのではない。余白のためにデザインする。
訪れた人は何もしなくてよい。ただ、そこにいればいい。

## 実装時の判断基準

何かを追加しようとするとき、必ずこの問いを立てる：

1. **これは必要か？** — 岡潔の言葉が届くために、本当に必要か。
2. **これは言葉を乱すか？** — 視線を言葉から奪うものではないか。
3. **これは静寂を破るか？** — ユーザーの時間を横取りしていないか。

「あってもいい」は理由にならない。「なければならない」だけが理由になる。

## やってはいけないこと

- アニメーションライブラリの導入
- ローディングスピナー・プログレス表示
- スキップ・再生・シェアなどのUI
- BGM・効果音
- アナリティクスの埋め込み
- フォントが読み込まれる前にテキストを表示すること

## 自己レビュー用チェックリスト

完成後、以下を確認する：

- [ ] 画面に余計なものが一切ないか
- [ ] フォントが確実に読み込まれてからシーンが始まるか
- [ ] アニメーションはCSSのみで実現されているか
- [ ] package.json に不要なライブラリが入っていないか
- [ ] 岡潔の言葉が、ページの主役になっているか
